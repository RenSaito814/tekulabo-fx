# テクラボFX - 要件定義

## サイト目的

FXコンテンツ事業の基盤ブログ。SEO記事でオーガニック流入を獲得し、アフィリエイト収益を得る。

## 対象読者

- FX初心者（これから始める人）
- FX学習中の個人トレーダー
- 自動売買・CFDに興味がある人

## コンテンツ要件

### 記事カテゴリ

| カテゴリ | 概要 |
|---------|------|
| FX基礎知識 | FXの仕組み、口座開設、ブローカー比較 |
| テクニカル分析 | ローソク足、インジケーター、チャートパターン |
| ファンダメンタルズ | 経済指標、FOMC、雇用統計 |
| リスク管理 | 損切り、ポジションサイズ、資金管理 |
| 自動売買 | EA、シストレ、バックテスト |

### 記事構成の方針

- FX初心者ロードマップ（fx-learning-roadmap）を軸に、段階的に学べる構成
- 各記事は相互に内部リンクで接続
- 全記事末尾にbroker-ctaショートコードと免責事項を配置

## 収益化要件

### ASP

- A8.net（登録済み、案件提携済み）

### アフィリエイト案件

| 案件 | 広告主 | 用途 |
|------|--------|------|
| DMM FX | DMM.com証券 | 初心者向け口座開設 |
| FXTF | ゴールデンウェイ・ジャパン | MT4・自動売買向け |
| シストレセレクト365 | フジトミ証券 | 選択型シストレ（くりっく365） |
| TOSSY | DMM.com証券 | マルチアセット投資アプリ |

### 将来的な収益化

- Google AdSense（記事数が増えてから申請）
- 他ASP（アクセストレード、afb、TCSアフィリエイト等）

- GitHub連携による自動デプロイ
