# テクラボFX - 設計判断の記録

## 2026-02 SSG選定: Hugo + PaperMod

- **決定**: Hugo（PaperModテーマ）を採用
- **理由**: 高速ビルド、SEO標準対応、日本語対応、設定がシンプル
- **代替案**: Next.js、Astro → 動的機能が不要なためオーバースペック
- **再検討済み（2026-03）**: 動的ランキング等は不要。手動キュレーションで十分。Hugoのまま継続。

## 2026-02 ホスティング: Cloudflare Pages

- **決定**: Cloudflare Pages（GitHub連携）
- **理由**: 無料、自動デプロイ、CDN、ドメインもCloudflare Registrarで管理済み

## 2026-03 アフィリエイト戦略: ハブ記事 + CTA方式

- **決定**: ブローカー比較記事（fx-broker-comparison）をハブとし、全記事にCTAショートコードで誘導
- **理由**: 個別記事にリンクを散らすよりも、比較記事に集約した方が更新管理が楽。CTAは自然な誘導になる。
- **構成**: broker-cta.htmlショートコード → 比較記事 → 各アフィリエイトリンク

## 2026-03 免責事項: ショートコード化

- **決定**: 免責事項をHugoショートコード（disclaimer.html）として全記事末尾に配置
- **理由**: 金融商品を扱うため法的に必須。一元管理で内容変更時に全記事に反映される。

## 2026-03 X API: pay-per-use方式

- **決定**: X API pay-per-use（従量課金）を利用予定
- **理由**: Legacy Free tierは2026年2月に廃止。1ツイート約$0.01、月$0.30程度で十分実用的。
- **状態**: Developer Portal復旧待ち
