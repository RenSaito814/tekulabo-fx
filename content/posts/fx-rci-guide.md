「RSIとストキャスティクス、どっちを使えばいいの？」と迷ったことはありませんか。実は、日本で特に人気のあるもう一つのオシレーター指標が**RCI（Rank Correlation Index / 順位相関係数）**です。

RCIは価格の「順位」に着目した指標で、[RSI](/posts/fx-technical-indicators/)や[ストキャスティクス](/posts/fx-stochastic-guide/)とは異なるアプローチで売買タイミングを判断できます。

[テクニカル分析の基礎](/posts/fx-technical-analysis-basics/)と[テクニカル指標の概要](/posts/fx-technical-indicators/)を読んでおくと、より理解しやすくなります。

---

## RCIとは

RCIは、**時間の順位**と**価格の順位**の相関を計算する指標です。スピアマンの順位相関係数をテクニカル分析に応用したもので、-100%から+100%の範囲で推移します。

### 基本的な考え方

RCIが測定するのは「時間が経つにつれて価格がどの程度規則的に上昇（または下降）しているか」です。

| RCIの値 | 意味 |
|---------|------|
| **+100%** | 完全な上昇（毎日価格が上昇） |
| **+80%以上** | 強い上昇トレンド → 買われすぎ |
| **0%付近** | 方向性なし |
| **-80%以下** | 強い下降トレンド → 売られすぎ |
| **-100%** | 完全な下降（毎日価格が下降） |

### RSI・ストキャスティクスとの違い

| 指標 | 計算の基準 | 範囲 | 特徴 |
|------|----------|------|------|
| [RSI](/posts/fx-technical-indicators/) | 値上がり幅と値下がり幅の比率 | 0〜100 | 価格変動の大きさを反映 |
| [ストキャスティクス](/posts/fx-stochastic-guide/) | 一定期間の値幅における現在位置 | 0〜100 | 直近の価格位置を反映 |
| **RCI** | 時間と価格の順位相関 | -100〜+100 | 値動きの「規則性」を反映 |

RCIの独自性は、価格の「大きさ」ではなく「順位」で計算する点です。これにより、急騰・急落の影響を受けにくく、滑らかな曲線を描きます。

## RCIの計算方法

計算式を暗記する必要はありませんが、仕組みを理解しておくと解釈が正確になります。

### 計算式

```
RCI = (1 - 6d / (n^3 - n)) x 100
```

| 要素 | 説明 |
|------|------|
| **n** | 期間（例: 9, 26, 52） |
| **d** | 時間順位と価格順位の差の2乗の合計 |

### 計算の手順

**ステップ1**: 期間内の各日に時間の順位をつける（最新日 = 1位）

**ステップ2**: 期間内の各日に価格の順位をつける（最高値 = 1位）

**ステップ3**: 各日の「時間順位 - 価格順位」の差を2乗して合計

**ステップ4**: 計算式に代入

### 計算例（5日間RCI）

| 日 | 時間順位 | 価格（円） | 価格順位 | 差 | 差の2乗 |
|----|---------|----------|---------|-----|---------|
| 5日前 | 5 | 148.50 | 4 | 1 | 1 |
| 4日前 | 4 | 148.20 | 5 | -1 | 1 |
| 3日前 | 3 | 149.00 | 3 | 0 | 0 |
| 2日前 | 2 | 149.80 | 2 | 0 | 0 |
| 1日前(最新) | 1 | 150.30 | 1 | 0 | 0 |

d = 1 + 1 + 0 + 0 + 0 = 2

RCI = (1 - 6 x 2 / (125 - 5)) x 100 = (1 - 12/120) x 100 = **90%**

最新日が最高値に近いため、RCIは高い値になっています。「規則的に上昇している」ことを示しています。

## 3本RCIの使い方

RCIの最も一般的な使い方は、**異なる期間のRCIを3本表示する**方法です。

### 期間設定

| ライン | 一般的な期間 | 役割 |
|--------|------------|------|
| **短期RCI** | 9 | エントリータイミング |
| **中期RCI** | 26 | トレンドの中間的な方向 |
| **長期RCI** | 52 | 大きなトレンドの方向 |

9・26・52は[一目均衡表](/posts/fx-ichimoku-guide/)の基本数値と同じです。日本発の指標同士で相性が良い設定とされています。

### 3本RCIの読み方

**強い上昇トレンド**:
- 長期RCI: +80%以上で上向き
- 中期RCI: +80%以上で上向き
- 短期RCI: 上下に振れながらも高い水準

**強い下降トレンド**:
- 長期RCI: -80%以下で下向き
- 中期RCI: -80%以下で下向き
- 短期RCI: 上下に振れながらも低い水準

**レンジ相場**:
- 長期RCI: 0%付近で横ばい
- 中期RCI: -80%〜+80%で上下
- 短期RCI: -80%〜+80%で頻繁に上下

## 売買シグナル

### 基本的なシグナル

**買いシグナル**:

| シグナル | 条件 | 信頼性 |
|---------|------|--------|
| 売られすぎからの反転 | 短期RCIが-80%以下から上向きに転じる | 中 |
| ゴールデンクロス | 短期RCIが中期RCIを下から上に抜ける | 中〜高 |
| 3本RCIの揃い | 3本すべてが上向きに転じる | 高 |

**売りシグナル**:

| シグナル | 条件 | 信頼性 |
|---------|------|--------|
| 買われすぎからの反転 | 短期RCIが+80%以上から下向きに転じる | 中 |
| デッドクロス | 短期RCIが中期RCIを上から下に抜ける | 中〜高 |
| 3本RCIの揃い | 3本すべてが下向きに転じる | 高 |

### トレンド方向でのフィルタリング

RCIのシグナルは、[ダウ理論](/posts/fx-dow-theory-guide/)や[移動平均線](/posts/fx-moving-average-guide/)で判断したトレンド方向と一致するもののみ採用すると、精度が大きく向上します。

- **上昇トレンド**: 買いシグナルのみ採用（押し目買い）
- **下降トレンド**: 売りシグナルのみ採用（戻り売り）

## RCIを使ったトレード戦略

### 戦略1: 押し目買い・戻り売り

トレンドフォローの基本戦略です。

**上昇トレンドでの押し目買い**:

1. 長期RCIが+80%付近で上向き（上昇トレンド確認）
2. 短期RCIが-80%付近まで低下（押し目形成）
3. 短期RCIが上向きに転じたらエントリー
4. [損切り](/posts/fx-stop-loss-techniques/)は直近の安値の下
5. 短期RCIが+80%に到達したら一部利益確定

### 戦略2: レンジ逆張り

長期RCIが横ばいのレンジ相場で使います。

1. 長期RCIが0%付近で横ばい（レンジ確認）
2. 短期RCIが-80%以下 → 買いエントリー
3. 短期RCIが+80%以上 → 売りエントリー
4. [ボリンジャーバンド](/posts/fx-bollinger-bands-guide/)のバンドタッチと組み合わせるとさらに効果的

### 戦略3: トレンド転換

上級者向けの手法です。

1. 長期RCIが+80%から下向きに転じる（上昇トレンドの終了兆候）
2. 中期RCIが+80%から下向きに転じる（追加確認）
3. 短期RCIが-80%方向へ急落
4. 3本すべてが下向きになったことを確認して売りエントリー

## 他の指標との組み合わせ

### 移動平均線との組み合わせ

[移動平均線](/posts/fx-moving-average-guide/)でトレンド方向を確認し、RCIでタイミングを計ります。

- 価格が200MAの上 + 短期RCIが-80%から反転 → 買い
- 価格が200MAの下 + 短期RCIが+80%から反転 → 売り

### 一目均衡表との組み合わせ

[一目均衡表](/posts/fx-ichimoku-guide/)とRCIは日本発の指標同士で、期間設定（9・26・52）も共通しています。

- 雲の上 + RCI買いシグナル → 強い買い
- 雲の下 + RCI売りシグナル → 強い売り
- 雲の中 → RCIのシグナルは見送り

### ADXとの組み合わせ

[ADX](/posts/fx-adx-guide/)でトレンドの強さを確認し、RCIでエントリーします。

- ADX > 25（トレンドあり）→ RCIの押し目・戻り戦略を使用
- ADX < 20（レンジ）→ RCIの逆張り戦略を使用

## ダイバージェンス

RCIでも[RSIと同様のダイバージェンス](/posts/fx-rsi-divergence/)が発生します。

### 強気のダイバージェンス（Bullish Divergence）

- 価格が安値を更新しているのに、RCIの安値が切り上がっている
- 下降トレンドの勢いが弱まっている兆候
- 反転上昇の可能性

### 弱気のダイバージェンス（Bearish Divergence）

- 価格が高値を更新しているのに、RCIの高値が切り下がっている
- 上昇トレンドの勢いが弱まっている兆候
- 反転下落の可能性

ダイバージェンスは単体ではなく、他の転換シグナル（[チャートパターン](/posts/fx-chart-patterns/)、[サポート・レジスタンス](/posts/fx-support-resistance-guide/)のブレイク）と組み合わせて判断してください。

## 時間足別の特徴

| 時間足 | 適した期間設定 | 用途 |
|--------|-------------|------|
| 5分〜15分足 | 9, 13, 26 | [スキャルピング](/posts/fx-scalping-guide/)（ダマシ多い） |
| 1時間足 | 9, 26, 52 | [デイトレード](/posts/fx-daytrading-guide/) |
| 4時間足 | 9, 26, 52 | [スイングトレード](/posts/fx-swing-trading-guide/) |
| 日足 | 9, 26, 52 | 中長期のポジション管理 |

[取引時間帯](/posts/fx-trading-sessions/)によってRCIの動き方も変わります。東京時間はレンジが多いため短期RCIが頻繁に上下し、ロンドン・NY時間はトレンドが出やすいため長期RCIの方向に注目します。

## RCIの注意点

### よくある失敗

1. **買われすぎ = 即売り**ではない
   - 強いトレンドでは+80%以上に張り付いたまま上昇が続く
   - 「買われすぎ」は「上昇トレンドが強い」という意味でもある

2. **短期RCIだけで判断する**
   - 短期RCIは頻繁にシグナルを出すため、ダマシが多い
   - 必ず中期・長期RCIと組み合わせて判断する

3. **すべてのシグナルでエントリーする**
   - トレンド方向と一致するシグナルのみ採用する
   - [リスク管理](/posts/fx-risk-management/)を徹底し、1回のトレードで資金の2%以上をリスクにさらさない

### RCIが苦手な場面

| 場面 | 理由 |
|------|------|
| 急騰・急落 | 順位計算のため、値動きの大きさが反映されにくい |
| 窓開け | 価格のギャップがあっても順位は変わらない |
| [経済指標](/posts/fx-economic-calendar-guide/)発表直後 | 急激な値動きに対してRCIの反応が遅い |

## まとめ

| ポイント | 内容 |
|---------|------|
| 基本機能 | 時間と価格の順位相関を-100〜+100で表示 |
| 標準設定 | 短期9、中期26、長期52の3本表示 |
| 主な使い方 | 押し目買い・戻り売り、レンジ逆張り |
| 強み | 滑らかな動き、ダマシが比較的少ない |
| 弱み | 急騰・急落への反応が遅い |

RCIは日本の個人トレーダーに根強い人気がある指標です。「順位」で計算するため価格のノイズに影響されにくく、3本表示にすることでトレンドの方向とエントリータイミングを同時に判断できます。

まずは[デモトレード](/posts/fx-demo-trading/)で3本RCIをチャートに表示し、過去の値動きとRCIの関係を観察するところから始めてみてください。[バックテスト](/posts/fx-backtesting/)で期間設定を検証し、[トレード日誌](/posts/fx-trading-journal-guide/)に結果を記録して、自分に合った使い方を見つけていきましょう。
