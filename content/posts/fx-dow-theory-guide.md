[テクニカル分析](/posts/fx-technical-analysis-basics/)を学ぶとき、最初に触れるべき理論が**ダウ理論（Dow Theory）**です。

19世紀末に生まれたこの理論は、現代のチャート分析やトレンドフォロー戦略の基盤となっています。[移動平均線](/posts/fx-moving-average-guide/)も[MACD](/posts/fx-macd-guide/)も[エリオット波動](/posts/fx-elliott-wave-guide/)も、元をたどればダウ理論の考え方に行き着きます。

この記事では、ダウ理論の6つの基本原則と、FXトレードへの実践的な応用方法を解説します。

---

## ダウ理論の歴史

ダウ理論は、**チャールズ・ダウ（Charles Dow）**が1900年前後に『ウォール・ストリート・ジャーナル』紙に寄稿した論説をもとに体系化されたものです。ダウ自身は理論として整理しておらず、死後にウィリアム・ハミルトンやロバート・レアらが「ダウ理論」としてまとめました。

チャールズ・ダウはダウ・ジョーンズ社の共同創業者であり、ダウ平均株価の考案者でもあります。元々は株式市場の分析のために考えられた理論ですが、その原則はFX市場を含むあらゆる金融市場に適用できます。

---

## ダウ理論の6つの基本原則

### 原則1: 市場はすべてを織り込む

価格には、経済指標、政治情勢、市場参加者の心理など、あらゆる情報がすでに反映されています。

つまり、[ファンダメンタルズ](/posts/fx-fundamentals/)で分析できる情報は、すでに価格に含まれているという考え方です。だからこそ、価格そのもの（チャート）を分析することに意味がある、というのがテクニカル分析の出発点です。

FXトレードにおいて、これは重要な示唆を与えてくれます。

- [経済指標](/posts/fx-economic-calendar-guide/)の発表前に、市場はすでに予想値を織り込んでいる
- サプライズ（予想と結果の乖離）が大きな値動きを生む
- 「なぜ動いたか」よりも「どう動いたか」に集中する

### 原則2: トレンドには3つの種類がある

ダウ理論では、トレンドを期間によって3つに分類します。

| 種類 | 期間の目安 | FXでの活用 |
|------|----------|-----------|
| **主要トレンド** | 1年〜数年 | 週足・月足で確認。長期の方向性 |
| **二次トレンド** | 3週間〜3ヶ月 | 日足で確認。[スイングトレード](/posts/fx-swing-trading-guide/)の基準 |
| **小トレンド** | 3週間未満 | 1時間足〜4時間足で確認。[デイトレード](/posts/fx-daytrading-guide/)の基準 |

これは[マルチタイムフレーム分析](/posts/fx-multi-timeframe/)そのものです。上位足で主要トレンドを確認し、下位足でエントリーのタイミングを計る手法は、ダウ理論の考え方を実践に落とし込んだものと言えます。

### 原則3: 主要トレンドには3つの局面がある

主要トレンドは以下の3つの局面を経て形成されます。

**上昇トレンドの場合**:

| 局面 | 特徴 | 参加者 |
|------|------|--------|
| **先行期（Accumulation）** | 一部の賢い投資家が買い始める。まだ一般には気づかれない | 機関投資家、プロトレーダー |
| **追随期（Public Participation）** | トレンドが明確になり、多くのトレーダーが参入 | 一般トレーダー |
| **利食い期（Distribution）** | 先行期に買った投資家が利益確定を始める。過熱感が出る | 初心者が参入、プロが撤退 |

FXトレーダーとして意識すべきことは以下の通りです。

- **先行期**を捉えるのは非常に難しい。無理に狙わない
- **追随期**がトレンドフォロー戦略で最も利益を得やすい局面
- **利食い期**は過熱感のサイン。新規エントリーは避ける
- [RSIのダイバージェンス](/posts/fx-rsi-divergence/)や出来高の変化が利食い期の手がかりになる

### 原則4: トレンドは明確な転換シグナルが出るまで継続する

ダウ理論で最も重要な原則です。

**上昇トレンドの定義**:
- 高値が切り上がり、安値も切り上がっている状態

**下降トレンドの定義**:
- 高値が切り下がり、安値も切り下がっている状態

**トレンド転換のシグナル**:
- 上昇トレンド中に安値が前の安値を下回った場合 → 上昇トレンドの終了の可能性
- 下降トレンド中に高値が前の高値を上回った場合 → 下降トレンドの終了の可能性

この考え方は、[サポート・レジスタンス](/posts/fx-support-resistance-guide/)や[ブレイクアウト戦略](/posts/fx-breakout-strategy/)の理論的な根拠でもあります。

### 原則5: トレンドは出来高でも確認される

ダウ理論では、トレンドの方向に動くときに出来高が増加し、反対方向に動くときに出来高が減少することを「健全なトレンド」の条件としています。

FXは分散市場のため正確な出来高データはありませんが、以下で代替できます。

| 代替指標 | 確認方法 |
|---------|---------|
| ティックボリューム | MT4/MT5で確認可能 |
| [ボラティリティ](/posts/fx-volatility-guide/) | ATRやボリンジャーバンド幅で代替 |
| [ADX](/posts/fx-adx-guide/) | トレンドの強さを数値で確認 |
| [通貨強弱](/posts/fx-currency-strength/) | 通貨ペア間の相対的な強弱 |

### 原則6: 平均は相互に確認される

ダウは、工業株平均と運輸株平均が同じ方向に動くことでトレンドを確認すべきだと主張しました。

FXへの応用としては、以下のような**相関分析**が該当します。

- ドル円が上昇 → ユーロドルが下落（ドル高の確認）
- 資源国通貨（豪ドル、カナダドル）が同方向に動く → 商品市場のトレンド確認
- 複数の[通貨ペア](/posts/fx-currency-pairs-guide/)で同じトレンドが確認できればシグナルの信頼性が高い

## 実践: ダウ理論によるトレンド判断

### 高値・安値の認識方法

ダウ理論を実践するには、チャート上の**高値（スイングハイ）**と**安値（スイングロー）**を正しく認識する必要があります。

**スイングハイの定義**:
- 前後のローソク足より高い高値をつけたポイント
- 少なくとも左右に数本の低いローソク足がある

**スイングローの定義**:
- 前後のローソク足より低い安値をつけたポイント
- 少なくとも左右に数本の高いローソク足がある

### トレンド判断の手順

**ステップ1**: チャート上の目立つ高値と安値をマーク

**ステップ2**: 高値と安値の推移を確認

| パターン | 判断 |
|---------|------|
| 高値切り上げ + 安値切り上げ | **上昇トレンド** |
| 高値切り下げ + 安値切り下げ | **下降トレンド** |
| 上記以外（高値切り上げ + 安値切り下げなど） | **トレンドなし（レンジ）** |

**ステップ3**: トレンド方向に沿ったトレードのみ行う

- 上昇トレンド → 買いのみ
- 下降トレンド → 売りのみ
- レンジ → トレードを見送るか、レンジ戦略を使う

## ダウ理論を活用したエントリー手法

### 押し目買い・戻り売り

ダウ理論に基づく最も基本的なエントリー手法です。

**上昇トレンドでの押し目買い**:

1. 高値切り上げ・安値切り上げの上昇トレンドを確認
2. 価格が一時的に下落（押し目）するのを待つ
3. 前回の安値付近、[サポートライン](/posts/fx-support-resistance-guide/)、または[フィボナッチ](/posts/fx-fibonacci-guide/)リトレースメントレベルで反発を確認
4. 反発を確認してからエントリー
5. 前回の安値の少し下に[損切り](/posts/fx-stop-loss-techniques/)を設定

**下降トレンドでの戻り売り**:

1. 高値切り下げ・安値切り下げの下降トレンドを確認
2. 価格が一時的に上昇（戻り）するのを待つ
3. 前回の高値付近やレジスタンスラインで反落を確認
4. 反落を確認してからエントリー
5. 前回の高値の少し上に損切りを設定

### トレンド転換を狙う手法

上級者向けの手法ですが、ダウ理論に基づくトレンド転換の判断は以下の通りです。

1. 上昇トレンド中に安値が前回の安値を下回る（最初の警告）
2. 続いて高値が前回の高値を超えられない（転換の確認）
3. 前回の安値のブレイクでエントリー

注意: トレンド転換を狙う手法はリスクが高いため、[リスク管理](/posts/fx-risk-management/)と[ポジションサイジング](/posts/fx-position-sizing-guide/)を徹底してください。

## ダウ理論と他の分析手法との関係

ダウ理論は多くのテクニカル分析手法の基盤となっています。

| 分析手法 | ダウ理論との関係 |
|---------|---------------|
| [エリオット波動](/posts/fx-elliott-wave-guide/) | ダウ理論のトレンド3局面を5波動+3波動に発展させた |
| [移動平均線](/posts/fx-moving-average-guide/) | トレンドの方向を数値的に表現 |
| [トレンドライン](/posts/fx-trend-line-guide/) | 高値・安値を線で結んでトレンドを視覚化 |
| [チャートパターン](/posts/fx-chart-patterns/) | トレンド転換・継続のパターンを体系化 |
| [ADX](/posts/fx-adx-guide/) | トレンドの「強さ」を定量化 |
| [パラボリックSAR](/posts/fx-parabolic-sar-guide/) | トレンド転換点を自動検知 |

## ダウ理論の限界

### 遅行性

ダウ理論はトレンドの「確認」に使う理論です。トレンドが確認できた時点で、すでにかなりの値幅が動いた後になります。先行期を捉えることは原理的にできません。

### 主観性

高値と安値の認識には主観が入ります。同じチャートを見ても、トレーダーによって判断が分かれることがあります。これを軽減するには以下が有効です。

- 明確なスイングハイ・スイングローのみを使う
- 複数の時間足で確認する
- [テクニカル指標](/posts/fx-technical-indicators/)で客観的に補完する

### レンジ相場への対応

ダウ理論はトレンド相場を前提とした理論です。レンジ相場では高値・安値の方向性が定まらず、判断が困難になります。レンジ相場が識別できた場合は、[ボリンジャーバンド](/posts/fx-bollinger-bands-guide/)や[ストキャスティクス](/posts/fx-stochastic-guide/)などのレンジ向き指標を使いましょう。

## まとめ

| ポイント | 内容 |
|---------|------|
| 本質 | 高値・安値の推移でトレンドを判断する |
| 最重要原則 | トレンドは明確な転換シグナルが出るまで継続する |
| 実践方法 | 高値切り上げ・安値切り上げ = 上昇トレンド |
| 活用法 | 押し目買い・戻り売りの根拠として使う |
| 限界 | 遅行性がある。レンジ相場では機能しにくい |

ダウ理論は100年以上前に生まれた理論ですが、その基本原則は現在のFX市場でも変わらず通用します。なぜなら、市場を動かしているのは人間の心理であり、それは時代が変わっても本質的には同じだからです。

まずは[デモトレード](/posts/fx-demo-trading/)で、チャート上の高値と安値を意識しながら実際のトレンド判断を練習してみてください。[トレード日誌](/posts/fx-trading-journal-guide/)にトレンド判断の根拠を記録する習慣をつけると、分析力が着実に向上します。
