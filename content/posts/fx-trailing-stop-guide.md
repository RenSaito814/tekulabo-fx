トレーリングストップは、**価格が有利な方向に動くにつれて損切りラインを自動的に引き上げる（引き下げる）手法**です。

通常の損切りはエントリー時に固定しますが、トレーリングストップは含み益が増えるほど損切り位置を有利な方向に移動させます。これにより、**利益を確保しながらトレンドに乗り続ける**ことができます。

この記事では、トレーリングストップの仕組みから具体的な設定方法までを解説します。

[損切りテクニック](/posts/fx-stop-loss-techniques/)と[リスク管理](/posts/fx-risk-management/)の知識が前提になります。

## トレーリングストップとは

### 基本的な仕組み

トレーリングストップの動作は以下の通りです（ロングポジションの場合）。

1. エントリー時に通常の損切りを設定する
2. 価格が上昇したら、損切りラインも一定のルールで引き上げる
3. 価格がさらに上昇したら、損切りラインもさらに引き上げる
4. 価格が反転して損切りラインに到達したら決済される

**重要なルール: 損切りラインは有利な方向にのみ移動する。不利な方向には絶対に動かさない。**

### 通常の損切りとの違い

| 項目 | 通常の損切り | トレーリングストップ |
|------|------------|-------------------|
| 損切り位置 | 固定 | 利益方向に移動 |
| 利益の伸ばし方 | 別途利確目標が必要 | 自動的にトレンドに追従 |
| 向いている相場 | どの相場でも基本 | トレンド相場 |
| リスク | エントリー時に確定 | 途中で縮小していく |

### トレーリングストップの具体例

USD/JPYを150.00でロング、損切り149.50（-50pips）の場合:

| 価格の動き | 損切り位置 | 含み益 | 確保された利益 |
|-----------|----------|--------|-------------|
| 150.00（エントリー） | 149.50 | 0 | なし（-50pipsリスク） |
| 150.50に上昇 | 150.00に引き上げ | +50pips | 損益ゼロ確保 |
| 151.00に上昇 | 150.50に引き上げ | +100pips | +50pips確保 |
| 151.50に上昇 | 151.00に引き上げ | +150pips | +100pips確保 |
| 151.00に反落 → 決済 | 151.00 | - | **+100pipsで確定** |

最高値151.50までの利益を全て取ることはできませんが、トレンドに乗りながら**+100pipsの利益を確保**できました。

## トレーリングストップの設定方法

### 方法1: 固定幅トレーリング

最もシンプルな方法です。価格から一定のpips数だけ離れた位置に損切りを置き、価格が動くたびに追従させます。

**設定例:**
- トレーリング幅: 50pips
- 価格が50pips上昇するごとに、損切りも50pips引き上げる

**メリット:**
- シンプルで分かりやすい
- MT4などの取引ツールに自動トレーリング機能がある

**デメリット:**
- ボラティリティを考慮しない。相場状況によって幅が適切でないことがある

### 方法2: ATRベーストレーリング

ATR（Average True Range）を使って、**現在のボラティリティに応じた幅**でトレーリングする方法です。

**設定例:**
- 損切り位置 = 直近高値 - ATR(14) x 2
- 新しい高値が更新されるたびに損切りを再計算

**メリット:**
- ボラティリティに自動的に適応する
- 高ボラティリティ時は幅が広くなり、不要な損切りを防げる
- 低ボラティリティ時は幅が狭くなり、利益を効率的に確保できる

**デメリット:**
- ATRの計算が必要（チャートツールで表示可能）

ATRについて詳しくは[ボラティリティ分析](/posts/fx-volatility-guide/)を参照してください。

### 方法3: 移動平均線トレーリング

[移動平均線](/posts/fx-moving-average-guide/)を損切りの基準にする方法です。

**設定例:**
- ロングポジション: 20EMAを下回ったら決済
- ショートポジション: 20EMAを上回ったら決済

**メリット:**
- トレンドの方向性と連動する
- 大きなトレンドに長く乗り続けやすい

**デメリット:**
- 移動平均線はやや遅行するため、反転時の利益の戻しが大きくなる

### 方法4: スイングポイントトレーリング

直近の安値（ロングの場合）または高値（ショートの場合）を損切り位置にする方法です。

**設定例（ロング）:**
- 新しいスイングロー（押し安値）が形成されるたびに、その少し下に損切りを移動
- 価格がその安値を割ったら、トレンド転換の可能性があるため決済

**メリット:**
- チャートの構造に基づいた合理的な設定
- [サポート・レジスタンス](/posts/fx-support-resistance-guide/)の考え方と整合性がある

**デメリット:**
- スイングポイントの判断に裁量が必要

### どの方法を使うべきか

| 方法 | 向いているスタイル |
|------|-----------------|
| 固定幅 | [スキャルピング](/posts/fx-scalping-guide/)、短期トレード |
| ATRベース | [デイトレード](/posts/fx-daytrading-guide/)全般 |
| 移動平均線 | [スイングトレード](/posts/fx-swing-trading-guide/)、大きなトレンド狙い |
| スイングポイント | スイングトレード、裁量トレーダー向け |

## トレードスタイル別の活用法

### デイトレードでの活用

デイトレードでは比較的タイトなトレーリングが適しています。

| 設定 | 値 |
|------|-----|
| 時間軸 | 15分足〜1時間足 |
| トレーリング方法 | ATR(14) x 1〜1.5 |
| 初期損切り | エントリーポイントから1.5〜2 ATR |

**ポイント:**
- その日のうちに決済するため、利益を積極的に確保する
- 損切りがエントリー価格を超えたら（損益ゼロ以上になったら）精神的な負担が大幅に減る

### スイングトレードでの活用

スイングトレードでは広めのトレーリングでトレンドに長く乗ります。

| 設定 | 値 |
|------|-----|
| 時間軸 | 4時間足〜日足 |
| トレーリング方法 | 20EMAまたはスイングポイント |
| 初期損切り | エントリーポイントから2〜3 ATR |

**ポイント:**
- 数日〜数週間保有するため、ある程度の含み益の戻しは許容する
- トレーリングが狭すぎると、通常の押し目で決済されてしまう

## よくある失敗と注意点

### 失敗1: トレーリングが狭すぎる

トレーリング幅を狭く設定しすぎると、通常の値動き（ノイズ）で頻繁に損切りされます。

**対策:** ATRの1倍以上の幅を確保する。少なくとも通常の押し目の深さよりも広くする。

### 失敗2: 損切りを不利な方向に動かす

含み益が減ると「もう少し戻るかもしれない」と損切りを元に戻したくなります。これは[トレード心理学](/posts/fx-trading-psychology/)で解説した「損失回避バイアス」です。

**ルール: 損切りは有利な方向にのみ動かす。絶対に元に戻さない。**

### 失敗3: レンジ相場で使用する

トレーリングストップはトレンド相場で力を発揮する手法です。レンジ相場では価格が上下に振れるため、トレーリングした損切りに引っかかりやすくなります。

**対策:** [移動平均線](/posts/fx-moving-average-guide/)でトレンドが出ていることを確認してからトレーリングストップを使う。レンジ相場では固定の利確目標を使う。

### 失敗4: 全額をトレーリングに委ねる

ポジション全体をトレーリングストップで管理すると、最高値付近での利益を逃しがちです。

**対策:** ポジションを分割して管理する。
- 半分: [ピボットポイント](/posts/fx-pivot-points/)のR1や[フィボナッチ](/posts/fx-fibonacci-guide/)の水準で利確（確実に利益を確保）
- 残り半分: トレーリングストップで利益を伸ばす

この「分割決済」のアプローチは、利益確保と利益拡大のバランスが取れた方法です。

## MT4/MT5でのトレーリングストップ設定

[MT4](/posts/fx-mt4-setup-guide/)にはトレーリングストップの自動機能が内蔵されています。

**設定手順:**
1. 保有ポジションを右クリック
2. 「トレーリングストップ」を選択
3. トレーリング幅を選択（ポイント単位）

**注意点:**
- MT4のトレーリングストップは**固定幅のみ**。ATRベースや移動平均線ベースは手動またはEA（自動売買プログラム）が必要
- MT4を閉じるとトレーリングストップが停止する。VPS等で常時稼働させるか、手動で管理する

## まとめ

| 項目 | ポイント |
|------|---------|
| 定義 | 利益方向に損切りを追従させる手法 |
| 基本ルール | 損切りは有利な方向にのみ動かす |
| 設定方法 | 固定幅、ATRベース、移動平均線、スイングポイント |
| 向いている相場 | トレンド相場。レンジ相場では不向き |
| 初心者向け | ATRベース（1.5〜2 ATR）が汎用的 |
| 分割決済 | 半分は固定利確、半分はトレーリングが効果的 |

トレーリングストップは「利益を伸ばす」ための技術です。[バックテスト](/posts/fx-backtesting/)で自分の戦略にトレーリングストップを組み込んだ場合の効果を検証し、[デモトレード](/posts/fx-demo-trading/)で実践してみましょう。
