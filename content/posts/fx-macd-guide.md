この記事では、MACDの構成要素を一つずつ掘り下げ、実践的なトレード判断にどう活かすかを詳しく解説します。

## MACDとは

MACD（Moving Average Convergence Divergence）は、日本語で「移動平均収束拡散法」と呼ばれます。1979年にジェラルド・アペルが開発した、**トレンドの方向と勢いを同時に把握できる**テクニカル指標です。

MACDは「遅行指標」に分類されますが、移動平均線そのものよりも早くトレンドの変化を捉えられるため、多くのトレーダーに使われています。

---

## MACDの構成要素

MACDは3つの要素で構成されます。

### 1. MACDライン

**短期EMA（12期間）から長期EMA（26期間）を引いた値**です。

```
MACDライン = EMA(12) - EMA(26)
```

- MACDラインがプラス → 短期EMAが長期EMAより上 → 上昇傾向
- MACDラインがマイナス → 短期EMAが長期EMAより下 → 下降傾向
- MACDラインがゼロに近い → 短期と長期がほぼ同じ → 方向感なし

EMA（指数平滑移動平均）はSMA（単純移動平均）よりも直近の価格に重みを置くため、価格変動への反応が早いという特徴があります。

### 2. シグナルライン

**MACDラインの9期間EMA**です。MACDラインを平滑化したもので、MACDラインとの交差がトレードシグナルになります。

```
シグナルライン = EMA(MACDライン, 9)
```

### 3. ヒストグラム

**MACDラインとシグナルラインの差**を棒グラフで表示したものです。

```
ヒストグラム = MACDライン - シグナルライン
```

- ヒストグラムがプラス → MACDラインがシグナルラインより上
- ヒストグラムがマイナス → MACDラインがシグナルラインより下
- ヒストグラムが拡大 → 勢いが増している
- ヒストグラムが縮小 → 勢いが弱まっている

ヒストグラムの拡大・縮小は、トレンドの「勢い」を視覚的に把握するのに役立ちます。

---

## MACDのデフォルト設定

| パラメータ | デフォルト値 | 役割 |
|-----------|------------|------|
| 短期EMA | 12 | 直近の価格トレンドを反映 |
| 長期EMA | 26 | より長い期間のトレンドを反映 |
| シグナル | 9 | MACDラインの平滑化 |

この設定は多くのトレーダーが使用しているため、これ自体が市場で意識されるラインになっています。パラメータを変更することも可能ですが、まずはデフォルトで使い込むことを推奨します。

## 基本的な売買シグナル

### ゴールデンクロス（買いシグナル）

MACDラインがシグナルラインを**下から上に**クロスしたとき、買いシグナルとなります。

**信頼度が高い条件:**

- ゼロラインより下でクロスが発生（底値圏からの反転）
- ヒストグラムがマイナスからプラスに転換
- クロスの角度が急（勢いがある）

### デッドクロス（売りシグナル）

MACDラインがシグナルラインを**上から下に**クロスしたとき、売りシグナルとなります。

**信頼度が高い条件:**

- ゼロラインより上でクロスが発生（天井圏からの反転）
- ヒストグラムがプラスからマイナスに転換
- クロスの角度が急

### ゼロラインクロス

MACDラインがゼロラインを上下にクロスすることも重要なシグナルです。

- ゼロラインを上抜け → 短期EMAが長期EMAを上回った（上昇トレンド入り）
- ゼロラインを下抜け → 短期EMAが長期EMAを下回った（下降トレンド入り）

ゼロラインクロスはゴールデンクロス/デッドクロスよりも遅れて発生しますが、トレンドの確認シグナルとして使えます。

## MACDダイバージェンス

ダイバージェンスとは、**価格の動きとMACDの動きが逆行する現象**です。[RSIダイバージェンス](/posts/fx-rsi-divergence/)と同じ考え方ですが、MACDでも強力なシグナルとなります。

### 強気のダイバージェンス（買いシグナル）

- 価格: 安値を切り下げている
- MACD: 安値を切り上げている

→ 下降トレンドの勢いが弱まっており、反転上昇の可能性を示唆

### 弱気のダイバージェンス（売りシグナル）

- 価格: 高値を切り上げている
- MACD: 高値を切り下げている

→ 上昇トレンドの勢いが弱まっており、反転下落の可能性を示唆

### ダイバージェンスの注意点

- ダイバージェンスは「勢いの低下」を示すだけで、即座に反転するとは限らない
- 強いトレンドではダイバージェンスが複数回発生してからようやく反転することもある
- 単独で使うのではなく、サポート・レジスタンスや他の指標と組み合わせて判断する

## ヒストグラムの活用法

ヒストグラムはクロスよりも早くトレンドの変化を捉えることができます。

### ヒストグラムの縮小に注目

ゴールデンクロスやデッドクロスが発生する前に、ヒストグラムは必ず縮小を始めます。この縮小を観察することで、クロスの発生を事前に予測できます。

**実践での使い方:**

1. ヒストグラムが連続して縮小している → クロスが近い
2. ヒストグラムがゼロを超えた → クロス発生
3. ヒストグラムが拡大し始めた → トレンドに勢いがある

### ヒストグラムの山と谷

ヒストグラムの山（プラスのピーク）や谷（マイナスのピーク）にも注目しましょう。

- 前回の山より低い山 → 上昇の勢いが弱まっている
- 前回の谷より浅い谷 → 下降の勢いが弱まっている

これはヒストグラム版のダイバージェンスとも言えます。

## MACDの弱点と対策

### 弱点1: レンジ相場でのダマシ

MACDはトレンドフォロー型の指標のため、レンジ相場では頻繁にクロスが発生し、ダマシのシグナルが増えます。

**対策:**

- [ボリンジャーバンド](/posts/fx-bollinger-bands-guide/)のバンド幅を見てレンジかトレンドかを判断
- ゼロライン付近での小さなクロスは無視する
- ADX（トレンドの強さを測る指標）と組み合わせる

### 弱点2: 遅行性

MACDは移動平均線をベースにしているため、トレンドの初動には遅れます。

**対策:**

- ヒストグラムの縮小を先行シグナルとして活用
- 短い時間足で確認して早めにエントリー（[マルチタイムフレーム分析](/posts/fx-multi-timeframe/)）
- [エントリーパターン](/posts/fx-entry-patterns/)とプライスアクションで補完

### 弱点3: 急激な値動きでの不安定さ

経済指標の発表時など、急激な値動きがあるとMACDが大きく振れて信頼性が低下します。

**対策:**

- [FOMC](/posts/fx-fomc-guide/)や[雇用統計](/posts/fx-nfp-trading-guide/)の前後はMACDのシグナルに頼らない
- 値動きが落ち着いてからシグナルを確認する

## 他の指標との組み合わせ

### RSIとの組み合わせ

MACDのクロス + RSIの水準を確認することで、シグナルの信頼度を上げられます。

- RSIが30付近から上昇し始めた局面でMACDゴールデンクロスが発生 → 強い買いシグナル
- RSIが70付近から低下し始めた局面でMACDデッドクロスが発生 → 強い売りシグナル

詳しくは[RSIダイバージェンス](/posts/fx-rsi-divergence/)も参照してください。

### 移動平均線との組み合わせ

MACDのシグナル方向と、長期移動平均線（200SMA）のトレンド方向が一致しているかを確認します。

- 200SMAが上向き + MACDゴールデンクロス → 大きなトレンドに乗った買い
- 200SMAが下向き + MACDデッドクロス → 大きなトレンドに乗った売り

詳しくは[移動平均線の実践ガイド](/posts/fx-moving-average-guide/)を参照してください。

## まとめ

MACDの実践ポイントを整理します。

| シグナル | 条件 | 信頼度を上げるポイント |
|---------|------|---------------------|
| ゴールデンクロス | MACDラインがシグナルラインを上抜け | ゼロライン以下で発生 |
| デッドクロス | MACDラインがシグナルラインを下抜け | ゼロライン以上で発生 |
| ダイバージェンス | 価格とMACDが逆行 | サポレジと一致 |
| ヒストグラム縮小 | 勢いの低下 | クロスの先行指標として |

MACDは万能ではありませんが、トレンドの方向と勢いを同時に把握できる便利な指標です。レンジ相場での弱点を理解した上で、他の指標と組み合わせて使いましょう。

自分のトレードルールにMACDを組み込む方法は[トレード戦略の作り方](/posts/fx-trading-strategy/)、資金管理については[リスク管理の基本](/posts/fx-risk-management/)を参照してください。
