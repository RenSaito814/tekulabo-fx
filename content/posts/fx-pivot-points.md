ピボットポイントは、**前日の高値・安値・終値から当日の重要な価格水準を自動的に算出する**テクニカル指標です。

多くの機関投資家やプロトレーダーが参考にしており、実際にこれらの水準で反発やブレイクアウトが発生しやすい傾向があります。

この記事では、ピボットポイントの計算方法から実践的な使い方までを解説します。

[サポート・レジスタンス](/posts/fx-support-resistance-guide/)と[デイトレード](/posts/fx-daytrading-guide/)の基本知識が前提になります。

---

## ピボットポイントとは

### 基本的な考え方

ピボットポイント（PP）は、前日の**高値（H）・安値（L）・終値（C）**の3つの値を使って算出される価格水準です。

この値を中心に、上下にサポートライン（S1、S2、S3）とレジスタンスライン（R1、R2、R3）が展開されます。

### なぜピボットポイントが機能するのか

ピボットポイントが有効に機能する理由は以下の通りです。

- **計算式が統一されている**: 世界中のトレーダーが同じ値を見ている
- **機関投資家も使用**: プロの売買判断にも組み込まれている
- **自己実現的予言**: 多くの参加者が同じ水準を意識するため、実際にその水準で反応が起きやすい

これは[サポート・レジスタンス](/posts/fx-support-resistance-guide/)と同じメカニズムです。

## ピボットポイントの計算方法

### 標準ピボット（フロアピボット）

最も一般的な計算方法です。

**ピボットポイント（PP）:**

PP = (H + L + C) / 3

**レジスタンスライン:**

| レベル | 計算式 |
|--------|--------|
| R1 | (2 x PP) - L |
| R2 | PP + (H - L) |
| R3 | H + 2 x (PP - L) |

**サポートライン:**

| レベル | 計算式 |
|--------|--------|
| S1 | (2 x PP) - H |
| S2 | PP - (H - L) |
| S3 | L - 2 x (H - PP) |

### 計算例

前日の値動きが以下の場合:

| 項目 | 値 |
|------|-----|
| 高値（H） | 150.50 |
| 安値（L） | 149.50 |
| 終値（C） | 150.20 |

PP = (150.50 + 149.50 + 150.20) / 3 = **150.07**

| レベル | 値 |
|--------|------|
| R3 | 151.63 |
| R2 | 151.07 |
| R1 | 150.63 |
| **PP** | **150.07** |
| S1 | 149.63 |
| S2 | 149.07 |
| S3 | 148.63 |

### 他のピボット計算方法

標準ピボット以外にもいくつかのバリエーションがあります。

| 種類 | 特徴 |
|------|------|
| ウッディ・ピボット | 終値に重みを置く。当日の始値付近での反応を重視 |
| カマリラ・ピボット | レベル間の幅が狭い。レンジ相場での逆張りに向いている |
| フィボナッチ・ピボット | [フィボナッチ比率](/posts/fx-fibonacci-guide/)（38.2%、61.8%）を使用 |

初心者は**標準ピボット**から始めれば十分です。

## ピボットポイントの見方

### PPの位置で相場の方向を判断する

当日の価格がPPに対してどこにあるかで、相場の方向感を判断できます。

| 価格の位置 | 意味 |
|-----------|------|
| PP より上で推移 | 強気（買い優勢） |
| PP より下で推移 | 弱気（売り優勢） |
| PP 付近で推移 | 方向感なし（様子見） |

### 各レベルの役割

| レベル | 役割 |
|--------|------|
| R1 | 最初のレジスタンス。ここで反落するケースが多い |
| R2 | 強いレジスタンス。R1を超えるとR2が次の目標 |
| R3 | 非常に強い上昇時のみ到達。通常は到達しない |
| S1 | 最初のサポート。ここで反発するケースが多い |
| S2 | 強いサポート。S1を割るとS2が次の目標 |
| S3 | 非常に強い下落時のみ到達。通常は到達しない |

一般的に、**1日の値動きはR1〜S1の範囲に収まることが多い**です。R2/S2やR3/S3に到達するのはボラティリティが高い日に限られます。

## ピボットポイントを使ったトレード戦略

### 戦略1: PPでの反転トレード

ピボットポイント（PP）は当日の「均衡点」として機能します。

**ロングの条件:**
1. 価格がPPまで下落する
2. PP付近でローソク足の反転パターン（[ピンバー](/posts/fx-candlestick-patterns/)等）が出現
3. 損切り: PPの少し下
4. 利確: R1

**ショートの条件:**
1. 価格がPPまで上昇する
2. PP付近でローソク足の反転パターンが出現
3. 損切り: PPの少し上
4. 利確: S1

### 戦略2: ブレイクアウトトレード

S1/R1などのレベルを[ブレイクアウト](/posts/fx-breakout-strategy/)する動きを捉えます。

**R1ブレイクアウト（ロング）:**
1. 価格がR1を終値で上抜ける
2. R1がサポートに転換したことを確認（リテスト）
3. エントリー
4. 損切り: R1の少し下
5. 利確: R2

### 戦略3: レンジトレード（S1〜R1）

多くの日はS1〜R1の範囲で値動きが収まります。

- S1付近で買い → R1付近で利確
- R1付近で売り → S1付近で利確
- 損切り: S2またはR2の少し向こう

この戦略は、**大きなトレンドが出ていないレンジ相場**で有効です。[移動平均線](/posts/fx-moving-average-guide/)が横ばいであることを確認してから使いましょう。

## 他のテクニカル指標との組み合わせ

### ピボット + 移動平均線

[移動平均線](/posts/fx-moving-average-guide/)でトレンド方向を確認し、ピボットでエントリーポイントを絞り込む組み合わせです。

- 移動平均線が上向き → ピボットのサポートレベル（PP、S1）での買いのみ
- 移動平均線が下向き → ピボットのレジスタンスレベル（PP、R1）での売りのみ

### ピボット + RSI

[RSI](/posts/fx-technical-indicators/)の過熱感とピボットの水準が重なるポイントは信頼性が高くなります。

- S1付近 + RSIが30以下 → 反発の可能性が高い
- R1付近 + RSIが70以上 → 反落の可能性が高い

### ピボット + フィボナッチ

[フィボナッチ・リトレースメント](/posts/fx-fibonacci-guide/)の水準とピボットのレベルが近い位置にある場合、その水準の信頼性が高まります。複数の根拠が重なる「コンフルエンス」は強力なシグナルです。

## ピボットポイントの注意点

### 時間軸の設定に注意

ピボットポイントの計算に使う「前日」は、**ニューヨーク市場のクローズ（日本時間7:00、夏時間6:00）**を基準にするのが一般的です。チャートツールによって基準時間が異なる場合があるため、設定を確認しましょう。

### 一方的なトレンド相場では機能しにくい

強いトレンドが出ている日は、S3やR3を超えて一方的に動くことがあります。ピボットポイントは**レンジ相場〜緩やかなトレンド**で最も機能します。

### 単独での使用を避ける

ピボットポイントは目安であり、**他のテクニカル指標や[マルチタイムフレーム分析](/posts/fx-multi-timeframe/)と組み合わせる**ことで精度が上がります。ピボットの水準だけを根拠にエントリーするのは避けましょう。

### 経済指標の影響

重要な[経済指標](/posts/fx-economic-calendar-guide/)の発表時は、テクニカルとは無関係に大きく動くことがあります。指標発表の前後はピボットの水準が機能しにくくなります。

## まとめ

| 項目 | ポイント |
|------|---------|
| 定義 | 前日の高値・安値・終値から当日の節目を算出 |
| 主要レベル | R3, R2, R1, PP, S1, S2, S3 |
| 方向判断 | PPの上=強気、下=弱気 |
| 通常の値幅 | S1〜R1に収まることが多い |
| 活用法 | 反転トレード、ブレイクアウト、レンジトレード |
| 注意点 | 強トレンド時は機能しにくい。他の指標と併用 |

ピボットポイントは計算式が明確で[バックテスト](/posts/fx-backtesting/)がしやすい指標です。まずは[デモトレード](/posts/fx-demo-trading/)でピボットの各レベルでの価格の反応を観察するところから始めましょう。
