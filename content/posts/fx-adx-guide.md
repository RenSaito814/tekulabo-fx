「トレンドが出ているのか、レンジなのかが分からない」――これはFX初心者が最も悩むポイントの一つです。

[移動平均線](/posts/fx-moving-average-guide/)や[MACD](/posts/fx-macd-guide/)はトレンドの**方向**を教えてくれますが、そのトレンドが**どれだけ強いのか**は教えてくれません。そこで役立つのが**ADX（Average Directional Index / 平均方向性指数）**です。

この記事では、ADXの仕組みから実践的な使い方まで解説します。

[テクニカル分析の基礎](/posts/fx-technical-analysis-basics/)と[テクニカル指標の概要](/posts/fx-technical-indicators/)を読んでおくと、より理解しやすくなります。

---

## ADXとは

ADXは、1978年にJ・ウェルズ・ワイルダー・ジュニア（J. Welles Wilder Jr.）が考案したテクニカル指標です。ワイルダーは[RSI](/posts/fx-technical-indicators/)やATR（[ボラティリティ分析](/posts/fx-volatility-guide/)参照）の開発者でもあります。

### ADXの基本的な考え方

ADXが測定するのは**トレンドの強さ**であり、**トレンドの方向ではありません**。

- ADXの値が高い → トレンドが強い（上昇でも下降でも）
- ADXの値が低い → トレンドが弱い（レンジ相場の可能性）

この特性がADXの最大の特徴です。上昇トレンドでも下降トレンドでも、トレンドが強ければADXは上昇します。

### 3本のライン

ADXは以下の3本のラインで構成されます。

| ライン | 名称 | 役割 |
|--------|------|------|
| **ADX** | 平均方向性指数 | トレンドの強さ（0〜100） |
| **+DI** | プラス方向性指標 | 上昇の強さ |
| **-DI** | マイナス方向性指標 | 下降の強さ |

+DIと-DIがトレンドの**方向**を、ADXがトレンドの**強さ**を示します。

## ADXの計算方法

ADXの計算は複数のステップを経ます。計算式を暗記する必要はありませんが、「何を測定しているか」を理解することで実践での判断力が上がります。

### ステップ1: 方向性の算出（+DM / -DM）

まず、価格の動きから上昇方向と下降方向の「方向性」を算出します。

```
+DM = 当日の高値 - 前日の高値（正の値の場合のみ）
-DM = 前日の安値 - 当日の安値（正の値の場合のみ）
```

- +DMが-DMより大きい場合: +DMを採用、-DM = 0
- -DMが+DMより大きい場合: -DMを採用、+DM = 0
- 両方ゼロ以下の場合: 両方 = 0

### ステップ2: 方向性指標（+DI / -DI）

方向性をATR（真の値幅）で割って正規化します。

```
+DI = (+DMの14期間平滑平均) / ATR(14) x 100
-DI = (-DMの14期間平滑平均) / ATR(14) x 100
```

これにより、+DIと-DIはそれぞれ0〜100の範囲になります。

### ステップ3: ADXの算出

+DIと-DIの乖離度合いからADXを計算します。

```
DX = |+DI - (-DI)| / (+DI + (-DI)) x 100
ADX = DXの14期間平滑平均
```

+DIと-DIの差が大きいほど（= 一方向への動きが明確なほど）ADXの値は高くなります。

一般的なパラメータは**14期間**です。

## ADXの読み方

### トレンド強度の判断基準

| ADXの値 | トレンド状態 | トレード戦略 |
|---------|------------|------------|
| 0〜20 | トレンドなし / 非常に弱い | レンジ戦略が有効。トレンドフォローは避ける |
| 20〜25 | トレンド発生の兆候 | 注目。エントリー準備 |
| 25〜50 | 明確なトレンド | トレンドフォロー戦略が有効 |
| 50〜75 | 非常に強いトレンド | トレンド継続を前提に利益を伸ばす |
| 75〜100 | 極端に強いトレンド | まれ。反転に注意 |

**実践上の目安**: ADXが**25を超えたらトレンドあり、20以下はレンジ**と判断するのが一般的です。ただし、20・25・50はあくまで一般的な目安であり、通貨ペアや時間足によって最適な閾値は異なります。[バックテスト](/posts/fx-backtesting/)で過去データに対する有効性を確認してから使いましょう。

### +DIと-DIの見方

+DIと-DIの位置関係でトレンドの方向が分かります。

| 状態 | 意味 |
|------|------|
| +DIが-DIの上 | 上昇圧力が優勢 |
| -DIが+DIの上 | 下降圧力が優勢 |
| +DIと-DIが接近 | 方向感が薄い |

### ADXの上昇と下降

| ADXの動き | 意味 |
|----------|------|
| ADX上昇 | トレンドが**強まっている** |
| ADX下降 | トレンドが**弱まっている**（反転ではなくレンジ移行の可能性） |
| ADXが横ばい | トレンド強度に変化なし |

**注意**: ADXが下降しても、すぐにトレンドが反転するわけではありません。「トレンドの勢いが弱まっている」という意味であり、レンジへの移行を示唆しています。

## ADXの基本的な売買シグナル

### DIクロス

+DIと-DIのクロスは、買い手と売り手の優位性の変化を示唆します。ただしADXが低い局面ではダマシが多く、単独では使いにくいシグナルです。

**買いシグナルの候補:**

- +DIが-DIを下から上にクロス（上昇圧力が下降圧力を上回った）
- ADXが25以上であること（トレンドが存在する確認）

**売りシグナルの候補:**

- -DIが+DIを下から上にクロス（下降圧力が上昇圧力を上回った）
- ADXが25以上であること

ADXが20以下の状態でのDIクロスはダマシが多く、シグナルとしての信頼性が低いとされています。

### ADX上昇 + DIクロスの組み合わせ

より信頼性の高いシグナルは、DIクロスとADXの上昇が同時に起きる場面です。

1. +DIが-DIを上にクロス（方向の確認）
2. ADXが上昇を始める（強さの確認）
3. ADXが25を超える（トレンド成立の確認）
4. 買いエントリーを検討

3つの条件が揃うことで、ダマシを減らすことができます。

## ADXを使ったトレード戦略

### 戦略1: トレンドフィルターとして使う

ADXの最も実践的な使い方は、**他の指標のシグナルをフィルタリングする**ことです。

**例: 移動平均線クロス + ADXフィルター**

[移動平均線クロス](/posts/fx-ma-cross-strategy/)は、レンジ相場ではダマシが多発します。ADXでフィルタリングすることで、トレンドが明確な場面でのみシグナルを採用できます。

- 移動平均線のゴールデンクロスが発生
- ADXが25以上 → エントリー
- ADXが20以下 → 見送り

この考え方は移動平均線に限らず、[MACD](/posts/fx-macd-guide/)、[ストキャスティクス](/posts/fx-stochastic-guide/)、[ブレイクアウト](/posts/fx-breakout-strategy/)など、あらゆるトレンドフォロー系シグナルに応用できます。

### 戦略2: レンジ相場の特定

ADXが20以下の状態は、レンジ相場である可能性が高いです。

- ADXが20以下で推移 → [サポート・レジスタンス](/posts/fx-support-resistance-guide/)間の逆張りが有効
- ADXが20以下から上昇を始める → レンジからトレンドへの転換に注目
- ADXが低下を続ける → トレンドの方向性が弱まっている。レンジ化や持ち合いへの移行に注意。[ボリンジャーバンド](/posts/fx-bollinger-bands-guide/)のスクイーズなど他指標と重なれば、[ブレイクアウト](/posts/fx-breakout-strategy/)準備局面の可能性

### 戦略3: トレンド終了の判断

ADXはトレンドの終了を判断するのにも役立ちます。

- ADXが50以上から下降を始める → トレンドの勢いが弱まっている
- ADXがピークをつけて下降 → ポジションの利益確定を検討
- ADXが25以下に低下 → トレンドフォローからレンジ戦略への切り替えを検討

ただし、ADXの下降は「トレンドが弱まっている」という意味であり、「反転する」ではない点に注意してください。ADX低下だけで即手仕舞いせず、直近高値・安値の更新失敗、DIの再クロス、移動平均線割れなど価格の動きと合わせて判断しましょう。

## 他のテクニカル指標との組み合わせ

ADXは単独よりも、他の指標との組み合わせで大きな効果を発揮します。

### 移動平均線 + ADX

[移動平均線](/posts/fx-moving-average-guide/)でトレンド方向を確認し、ADXでトレンド強度を確認する組み合わせです。

以下は一般的な手法の一例です。

**押し目買いの条件:**

1. 価格が20日移動平均線の上にある（上昇トレンド）
2. ADXが25以上（トレンドが強い）
3. +DIが-DIの上にある（上昇圧力が優勢）
4. 価格が移動平均線まで押した後に反発
5. 買いエントリーを検討

### ボリンジャーバンド + ADX

[ボリンジャーバンド](/posts/fx-bollinger-bands-guide/)のスクイーズ（バンド収縮）とADXの低下が重なると、ブレイクアウトの前兆になることがあります。

- ボリンジャーバンドがスクイーズ + ADXが20以下 → ブレイクアウト待ち
- ブレイクアウト発生 + ADXが上昇開始 → エントリー検討

### RSI + ADX

[RSI](/posts/fx-rsi-divergence/)は、トレンド相場では買われすぎ・売られすぎの水準に張り付き続けることがあります。ADXでトレンドの有無を確認することで、RSIの使い方を切り替えられます。

| ADXの状態 | RSIの使い方 |
|----------|-----------|
| ADX 20以下（レンジ） | 70以上で売り、30以下で買いの逆張り |
| ADX 25以上（トレンド） | ダイバージェンスのみに注目。逆張りは控える |

### フィボナッチ + ADX

ADXが25以上の状態でトレンドが確認されている場面では、[フィボナッチ・リトレースメント](/posts/fx-fibonacci-guide/)の38.2%〜61.8%への押し戻りが良いエントリーポイントになります。ADXが低い状態ではフィボナッチ水準の信頼性も下がります。

## ADXのパラメータ設定

### 標準設定（14期間）

ワイルダーが提唱したデフォルト設定で、多くのチャートツールで初期値になっています。デイトレードからスイングトレードまで幅広く対応します。

### トレードスタイル別の調整

| 設定 | 特徴 | 用途 |
|------|------|------|
| 短い期間（7〜10） | 反応が速い。ノイズが多い | [スキャルピング](/posts/fx-scalping-guide/) |
| 標準（14） | バランスが良い | デイトレード・スイング |
| 長い期間（20〜30） | 反応が遅い。安定した読み取り | [スイングトレード](/posts/fx-swing-trading-guide/)・長期 |

パラメータを変更した場合は、[バックテスト](/posts/fx-backtesting/)で有効性を検証してから実戦に使いましょう。

## ADXの注意点

### トレンドの方向は分からない

ADXが上昇しても、それが上昇トレンドなのか下降トレンドなのかはADX単体では判断できません。方向は+DIと-DIの位置関係、または[移動平均線](/posts/fx-moving-average-guide/)や[トレンドライン](/posts/fx-trend-line-guide/)で確認してください。

### 反応が遅い

ADXは複数の平滑化処理を経て算出されるため、**他の指標に比べて反応が遅い**のが弱点です。トレンド発生の初動ではADXがまだ低い状態のことが多く、ADXが25を超えた時点ではすでに値幅の一部を逃している可能性があります。

対策:
- ADXの「上昇方向」に注目する（値が25に達する前でも、上昇を始めたことでトレンド発生を早期に察知できる）
- DIクロスをエントリーシグナルとして使い、ADXは確認用にする

### レンジ相場では機能しにくい

ADXが20以下でDIのクロスが頻発する場面では、シグナルの大半がダマシになります。ADXが低い時はトレンドフォロー系のシグナルを全て見送るのが賢明です。

### ADXの「ダマシ」

ADXが一時的に25を超えてすぐに戻るケースがあります。対策として以下が有効です。

- ADXが25を超えた状態が**数本（2〜3本）続く**ことを確認する
- ADXだけでなく、DIの位置関係やローソク足の動きも総合的に判断する
- [マルチタイムフレーム分析](/posts/fx-multi-timeframe/)で上位足のトレンド方向と一致しているか確認する

### 損切りは必ず設定する

ADXの読みが正しくても、相場が予想通りに動く保証はありません。[損切り](/posts/fx-stop-loss-techniques/)は直近の高値・安値やATR（[ボラティリティ分析](/posts/fx-volatility-guide/)参照）を基準に必ず設定してください。[ポジションサイジング](/posts/fx-position-sizing-guide/)も通常通り口座資金の1〜2%以内のリスクに抑えます。

## まとめ

| ポイント | 内容 |
|---------|------|
| ADXとは | トレンドの強さ（方向ではない）を0〜100で示す指標 |
| 3本のライン | ADX（強さ）、+DI（上昇）、-DI（下降） |
| 基本判断 | 25以上 = トレンドあり、20以下 = レンジ |
| 最大の活用法 | 他の指標のトレンドフィルターとして使う |
| 弱点 | 反応が遅い。方向は+DI/-DIや他の指標で確認が必要 |

ADXは「今トレンドフォロー戦略を使うべきか、レンジ戦略を使うべきか」の判断を助ける実用的な指標です。まずは[デモトレード](/posts/fx-demo-trading/)でチャートにADXを表示し、ADXの値と実際の値動きの関係を観察してみましょう。慣れてきたら[移動平均線クロス](/posts/fx-ma-cross-strategy/)やブレイクアウトのフィルターとして実践し、[トレード日誌](/posts/fx-trading-journal-guide/)に記録して効果を検証してください。
