FXを始めるとき、「何ロットで取引すればいいのか」は初心者が最初にぶつかる壁の一つです。

ロット数を適当に決めてしまうと、想定以上の損失を出したり、逆にリスクを取らなすぎて練習にならなかったりします。この記事では、ロットの基本概念から適切なロット数の計算方法まで解説します。

---

## ロット（Lot）とは

ロットとは、FXにおける**取引数量の単位**です。

### ロットの種類

| 名称 | 通貨量 | 1pip動いた時の損益（ドル円の場合） |
|------|--------|--------------------------------|
| 1ロット（1枚） | 10万通貨 | 約1,000円 |
| 0.1ロット（ミニロット） | 1万通貨 | 約100円 |
| 0.01ロット（マイクロロット） | 1,000通貨 | 約10円 |

※ブローカーによって1ロットの定義が異なる場合があります。国内ブローカーでは1ロット=1万通貨としているところもあるため、取引前に必ず確認してください。

### 具体的なイメージ

ドル円が150円のとき:

- **0.01ロット（1,000通貨）**: 15万円分の取引。1pip = 約10円
- **0.1ロット（1万通貨）**: 150万円分の取引。1pip = 約100円
- **1ロット（10万通貨）**: 1,500万円分の取引。1pip = 約1,000円

レバレッジがあるため、口座に1,500万円なくても1ロットの取引は可能です。しかし、ロットが大きいほど損益の振れ幅も大きくなります。

---

## レバレッジとロットの関係

レバレッジとは、**口座資金の何倍の取引ができるか**を示す倍率です。詳しくは[FXとは？](/posts/fx-basics-what-is-fx/)で解説しています。

### 必要証拠金の計算

```
必要証拠金 = 取引数量 x 現在のレート / レバレッジ
```

**例:** ドル円150円、0.1ロット（1万通貨）、レバレッジ25倍の場合

```
必要証拠金 = 10,000 x 150 / 25 = 60,000円
```

### レバレッジが高い = ロットを大きくできる ≠ 大きくすべき

レバレッジが高いと少ない資金で大きなロットを持てますが、それは同時にリスクも大きくなるということです。レバレッジの上限ではなく、**リスク管理の観点からロットを決める**ことが重要です。

## 適切なロット数の決め方

### ステップ1: 1トレードのリスク額を決める

[リスク管理の基本](/posts/fx-risk-management/)で学んだ通り、1トレードのリスクは口座資金の**1〜2%以内**に抑えます。

| 口座資金 | リスク1% | リスク2% |
|---------|---------|---------|
| 10万円 | 1,000円 | 2,000円 |
| 30万円 | 3,000円 | 6,000円 |
| 50万円 | 5,000円 | 10,000円 |
| 100万円 | 10,000円 | 20,000円 |

### ステップ2: 損切り幅を決める

[損切りの技術](/posts/fx-stop-loss-techniques/)で学んだ通り、テクニカル根拠に基づいて損切り幅を決めます。

### ステップ3: ロット数を計算する

```
ロット数 = リスク額 / (損切り幅pips x 1pipあたりの損益)
```

**計算例:** 口座資金30万円、リスク1%（3,000円）、損切り30pips、ドル円の場合

```
ロット数 = 3,000 / (30 x 100) = 1万通貨 = 0.1ロット
```

**計算例2:** 口座資金30万円、リスク1%（3,000円）、損切り50pips、ドル円の場合

```
ロット数 = 3,000 / (50 x 100) = 0.6万通貨 = 0.06ロット
```

損切り幅が広くなるほど、ロット数は小さくなります。これが正しい姿です。

※上記の計算例はドル円（対円通貨ペア）の場合です。ユーロドルなどドルストレートでは1pipあたりの損益が異なります。通貨ペアごとのpip値の違いについては[通貨ペアガイド](/posts/fx-currency-pairs-guide/)を参照してください。

## 口座資金別のロット目安

初心者向けの目安をまとめます（リスク1%、損切り30pips、ドル円の場合）。

| 口座資金 | ロット数 | 通貨量 | 30pips逆行時の損失 |
|---------|---------|--------|-------------------|
| 5万円 | 0.01〜0.02 | 1,000〜2,000 | 300〜600円 |
| 10万円 | 0.03 | 3,000 | 900円 |
| 30万円 | 0.1 | 10,000 | 3,000円 |
| 50万円 | 0.16 | 16,000 | 4,800円 |
| 100万円 | 0.33 | 33,000 | 9,900円 |

※あくまで目安です。損切り幅が変われば適切なロットも変わります。

## 初心者がやりがちなロットの失敗

### 失敗1: 口座資金に対してロットが大きすぎる

「早く稼ぎたい」という気持ちからロットを上げすぎると、数回の連敗で口座資金の大半を失います。<a href="https://px.a8.net/svt/ejp?a8mat=4AZA43+4784FM+1WP2+6BMG2" rel="nofollow">DMM FX</a><img border="0" width="1" height="1" src="https://www16.a8.net/0.gif?a8mat=4AZA43+4784FM+1WP2+6BMG2" alt="">のミニ通貨ペア（1,000通貨）や、<a href="https://px.a8.net/svt/ejp?a8mat=4AZA44+2S0UEQ+4SM6+61C2Q" rel="nofollow">松井証券 MATSUI FX</a><img border="0" width="1" height="1" src="https://www14.a8.net/0.gif?a8mat=4AZA44+2S0UEQ+4SM6+61C2Q" alt="">のように1通貨（約100円）から取引できるブローカーで、小さく始めることが重要です。

- 口座10万円で1ロット（10万通貨）→ 10pips逆行で1万円の損失（資金の10%）
- 3連敗で資金の30%が消える

### 失敗2: 毎回同じロットで取引する

損切り幅が20pipsの時も50pipsの時も同じロットで取引すると、リスクが一定になりません。損切り幅に応じてロットを調整するのが正しいリスク管理です。

詳しくは[ポジションサイズ計算の実践ガイド](/posts/fx-position-sizing-guide/)を参照してください。

### 失敗3: 含み損が出たらロットを増やす（ナンピン）

含み損のポジションに追加でエントリーして平均取得価格を下げる手法（ナンピン）は、初心者には非常に危険です。想定と逆に動き続けた場合、損失が加速的に膨らみます。

## まとめ

| ポイント | 内容 |
|---------|------|
| ロットの計算 | リスク額 / (損切り幅 x 1pip損益) で決める |
| リスク上限 | 口座資金の1〜2%以内 |
| 損切り幅で調整 | 損切りが広い → ロットを小さく |
| 初心者の心得 | 小さいロットから始めて徐々に慣れる |

ロットの決め方はリスク管理の根幹です。[デモトレード](/posts/fx-demo-trading/)の段階からこの計算を習慣化し、[リアルトレード](/posts/fx-real-trading/)への移行に備えましょう。
