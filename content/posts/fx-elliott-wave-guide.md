「なぜ相場はジグザグに動くのか」「今の上昇はどこまで続くのか」――こうした疑問に一つの枠組みを与えてくれるのが**エリオット波動理論**です。

この記事では、エリオット波動の基本構造から実践での使い方、そして理論の限界まで解説します。

[テクニカル分析の基礎](/posts/fx-technical-analysis-basics/)、[トレンドライン](/posts/fx-trend-line-guide/)、[フィボナッチ・リトレースメント](/posts/fx-fibonacci-guide/)の知識があると理解しやすくなります。

---

## エリオット波動理論とは

エリオット波動理論は、1930年代にラルフ・ネルソン・エリオット（Ralph Nelson Elliott）が提唱した相場分析の理論です。

核心となる考え方はシンプルです。

**相場は「推進波（5波）」と「修正波（3波）」の繰り返しで動く。**

エリオットは、株式市場の過去データを徹底的に分析し、値動きには群集心理に基づく一定のパターン（波動）が存在することを発見しました。この理論はその後FXや商品先物など、あらゆる金融市場に応用されています。

---

## 基本構造: 推進5波と修正3波

エリオット波動の基本は**8つの波**で1サイクルを形成する構造です。

### 推進波（Impulse Wave）: 5つの波

トレンド方向に進む5つの波です。上昇トレンドの場合:

```
        第3波
         /\
        /  \    第5波
第1波  /    \    /\
  /\  /      \  /  \
 /  \/        \/    \
/   第2波    第4波    \
```

| 波 | 方向 | 特徴 |
|----|------|------|
| **第1波** | トレンド方向 | 新しいトレンドの始まり。多くのトレーダーはまだ気づいていない |
| **第2波** | 調整（逆行） | 第1波の利益確定による押し。第1波の始点を下回らない |
| **第3波** | トレンド方向 | 最も力強い波。多くの参加者がトレンドに気づいて参入する |
| **第4波** | 調整（逆行） | 第3波の利益確定。第1波の高値の価格帯に重ならない |
| **第5波** | トレンド方向 | トレンドの最終局面。新規参入者が遅れて飛び乗る |

### 修正波（Corrective Wave）: 3つの波

推進波の後に続く、トレンドとは逆方向の3つの波です。A・B・Cで表記します。

```
第5波の高値
  \
   \ A波
    \
     \  B波
      \/\
         \
          \ C波
           \
```

| 波 | 方向 | 特徴 |
|----|------|------|
| **A波** | 逆行 | 利益確定や反転の始まり |
| **B波** | 一時的な戻り | 「まだトレンドは続く」と考える人の買い |
| **C波** | 逆行 | 本格的な調整。A波と同程度かそれ以上の値幅 |

### 1サイクルの全体像

推進5波 + 修正3波 = **8波で1サイクル**が完成します。このサイクルが完了すると、次のサイクルが始まります。

## エリオット波動の3つの絶対ルール

エリオット波動には**破ってはいけない3つのルール**があります。これを満たさない波のカウントは無効です。

### ルール1: 第2波は第1波の始点を超えて逆行しない

第2波の安値（上昇トレンドの場合）が第1波の始点を下回った場合、それは第2波ではありません。

### ルール2: 第3波は推進波の中で最も短くならない

第3波は第1波・第5波と比較して、値幅が最も小さくなることはありません。多くの場合、第3波が**最も長い**波になります。

### ルール3: 第4波は第1波の価格帯に重ならない

第4波の安値（上昇トレンドの場合）が第1波の高値を下回ることはありません。両者の価格帯が重なった場合、波のカウントを見直す必要があります。

### ガイドライン（絶対ではないが傾向として）

ルールとは別に、よく観察される傾向（ガイドライン）もあります。

| ガイドライン | 内容 |
|-------------|------|
| 交替の法則 | 第2波と第4波は異なるパターンになりやすい（第2波が急なら第4波は緩やか） |
| 第3波の延長 | FX市場では第3波が延長（他の推進波の1.618倍以上）することが多い |
| チャネリング | 推進波は平行チャネル内に収まりやすい |
| 第5波の特徴 | 第5波はモメンタム（勢い）が弱まることが多く、[RSIダイバージェンス](/posts/fx-rsi-divergence/)が発生しやすい |

## フィボナッチとエリオット波動

エリオット波動理論と[フィボナッチ](/posts/fx-fibonacci-guide/)は密接に関連しています。各波の値幅はフィボナッチ比率で予測できることが多いです。

### 推進波のフィボナッチ目安

| 波 | フィボナッチ比率の目安 |
|----|---------------------|
| **第2波** | 第1波の50%〜61.8%を戻すことが多い |
| **第3波** | 第1波の161.8%〜261.8%の値幅になりやすい |
| **第4波** | 第3波の23.6%〜38.2%を戻すことが多い |
| **第5波** | 第1波と同程度、または第1波の61.8%の値幅 |

### 修正波のフィボナッチ目安

| 波 | フィボナッチ比率の目安 |
|----|---------------------|
| **A波** | 第5波の値幅の61.8%〜100% |
| **B波** | A波の50%〜78.6%を戻す |
| **C波** | A波と同程度、またはA波の1.618倍 |

### 計算例

第1波が100pips上昇した場合:

- **第2波の下落**: 50〜61.8pips（第1波の50%〜61.8%）
- **第3波の上昇**: 161.8〜261.8pips（第1波の161.8%〜261.8%）
- **第4波の下落**: 第3波の値幅の23.6%〜38.2%

これらはあくまで目安です。フィボナッチ比率に「必ず一致する」わけではありませんが、波のカウントが正しいかを検証する材料になります。

## 波の階層構造（フラクタル）

エリオット波動の重要な特徴は、**波の中にさらに小さな波が含まれる**というフラクタル構造です。

- 日足の第3波を4時間足で見ると、その中に5つの小さな推進波がある
- 4時間足の各波を1時間足で見ると、さらに細かい波動構造がある

この考え方は[マルチタイムフレーム分析](/posts/fx-multi-timeframe/)と共通しています。

### 実践での活用

1. **上位足**で大きな波動のどこにいるかを把握する（例: 日足の第3波の途中）
2. **下位足**でエントリーポイントを探す（例: 4時間足の小さな第2波の終わり）

上位足で第3波が始まったと判断できれば、下位足で押し目を拾うことで、最も値幅が出やすい局面でトレードできます。

## 修正波のパターン

修正波（A-B-C）にはいくつかの典型的なパターンがあります。

### ジグザグ（Zigzag）

最もシンプルな修正パターンです。A-B-Cがそれぞれ鋭く動きます。

- A波: 5波構造
- B波: 3波構造（A波の50%〜78.6%を戻す）
- C波: 5波構造（A波と同程度かそれ以上の値幅）

**特徴**: 値幅が大きく、比較的短期間で修正が完了する。

### フラット（Flat）

横ばいに近い修正パターンです。

- A波: 3波構造
- B波: 3波構造（A波の始点付近まで戻す）
- C波: 5波構造

**特徴**: ジグザグと比べて値幅が小さい。レンジ相場に見える。

### トライアングル（Triangle）

[三角持ち合い](/posts/fx-chart-patterns/)の形をとる修正パターンです。a-b-c-d-eの5つの波で構成されます。

**特徴**: 第4波やB波で出現しやすい。ブレイクアウト後にトレンドが再開する。

## FXでのエリオット波動の使い方

### ステップ1: トレンドの方向を確認する

まず[トレンドライン](/posts/fx-trend-line-guide/)や[移動平均線](/posts/fx-moving-average-guide/)でトレンドの方向を確認します。エリオット波動はトレンドが明確な相場で機能しやすいです。

### ステップ2: 現在の波をカウントする

チャートを見て「今、何波目にいるのか」を判断します。

**カウントのコツ**:
- 3つの絶対ルールに違反していないか確認する
- [フィボナッチ・リトレースメント](/posts/fx-fibonacci-guide/)で各波の戻り幅が目安に合っているか確認する
- 上位足から下位足へ順に確認する

### ステップ3: 狙うべき波を選ぶ

| 波 | トレード適性 | 理由 |
|----|------------|------|
| 第1波 | 難しい | トレンド転換の初動で判断が難しい |
| **第3波** | **最適** | 最も値幅が大きく、方向が明確 |
| 第5波 | 注意が必要 | トレンドの最終局面。ダイバージェンスに注意 |
| C波 | 中級者向け | 修正の最終局面で逆張り（リスク高） |

初心者は**第3波に注目するアプローチ**が一般的です。

### ステップ4: エントリーとリスク管理

以下は一般的な手法の一例です。

**第3波を狙うエントリー例（上昇トレンド）:**

1. 第1波の上昇を確認する
2. 第2波の調整を待つ（第1波の50%〜61.8%の戻し）
3. フィボナッチ水準 + [サポートライン](/posts/fx-support-resistance-guide/)で反発を確認
4. 買いエントリー
5. 損切り: 第1波の始点の少し下
6. 利確: 第1波の161.8%エクステンション水準

[ポジションサイジング](/posts/fx-position-sizing-guide/)は通常通り、口座資金の1〜2%以内のリスクに抑えます。

## エリオット波動の限界と注意点

### リアルタイムでの波のカウントは難しい

エリオット波動の最大の課題は、**リアルタイムで正確に波をカウントすることが非常に難しい**という点です。

- 同じチャートを見ても、分析者によってカウントが異なることがある
- 相場が進むにつれてカウントの修正が必要になることがある
- 「後から見れば綺麗な波動だった」というケースが多い

### 単独での使用は避ける

エリオット波動だけでトレード判断を行うのはリスクが高いです。以下の指標と組み合わせて根拠を増やしましょう。

| 組み合わせる指標 | 用途 |
|----------------|------|
| [フィボナッチ](/posts/fx-fibonacci-guide/) | 各波の目標値・戻り幅の検証 |
| [移動平均線](/posts/fx-moving-average-guide/) | トレンドの確認、サポート・レジスタンス |
| [RSI](/posts/fx-rsi-divergence/) | 第5波のダイバージェンス検出 |
| [MACD](/posts/fx-macd-guide/) | モメンタムの変化を確認 |
| [ボリンジャーバンド](/posts/fx-bollinger-bands-guide/) | ボラティリティの変化と波の転換 |

### 全ての相場に適用できるわけではない

- レンジ相場では波のカウントが曖昧になりやすい
- 経済指標発表時の急変動はエリオット波動で予測できない
- 流動性が低い時間帯ではノイズが多く、波のカウントが困難

### 完璧を目指さない

エリオット波動の全ての波を正確にカウントしようとすると、分析に時間がかかりすぎます。実用的には以下の使い方が現実的です。

- **大きな流れの中で「今どのあたりにいるか」を大まかに把握する**
- **第3波の可能性が高い場面を重点的に狙う**
- カウントに自信がない場面ではトレードしない

## まとめ

| ポイント | 内容 |
|---------|------|
| 基本構造 | 推進5波 + 修正3波 = 1サイクル |
| 3つの絶対ルール | 第2波は第1波始点を超えない / 第3波は最短にならない / 第4波は第1波と重ならない |
| フィボナッチ | 各波の値幅はフィボナッチ比率で予測できることが多い |
| 狙うべき波 | 初心者は第3波（最も力強い波）に集中 |
| 限界 | リアルタイムのカウントは難しい。他の指標と必ず併用する |

エリオット波動は相場の全体像を把握するための「地図」のような理論です。完璧なカウントよりも、「大局的に今どの局面にいるのか」を意識することが実践では重要です。

まずは[デモトレード](/posts/fx-demo-trading/)で過去チャートの波動をカウントする練習を行い、[トレード日誌](/posts/fx-trading-journal-guide/)に記録して精度を高めていきましょう。
